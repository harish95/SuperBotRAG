# Enterprise RAG Frontend

## Run locally

```bash
npm install
npm run dev
```

Create a `.env` file in `frontend/` if needed:

```bash
VITE_API_URL=http://localhost:8000
```

## Stack

- React + Vite
- TypeScript
- TailwindCSS
- React Router
- Axios
- TanStack Query
- Zustand
- shadcn-style UI components
- Lucide icons
